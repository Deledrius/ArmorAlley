Network Options

  • Game Style

    Play cooperatively with a best friend, or against your greatest foe.

  • Battle Selection

    Choose a battle designed for network play, or one of the original classics.
    Battles have different features, depending on difficulty:

    • Bullets: when "off," helicopters fire powerful "dumb" missiles.
    • Napalm: bombs explode wider, vs. a "spark" on the ground.
    • Stealth: enemy helicopters don't show on radar.
    • Jamming: radar visibility is impacted at all times.
    • Fairness: battlefield advantage, left (-) to right (+). 0 = balanced.
    • IQ: A measure of enemy "AI", between tactics and abilities.

    "Fair and balanced"

    For PvP games, try a battle with F = 0 (e.g., "Cavern Cobra"), where players own half the field to start. If you like a challenge, pick one with a bias to your opponent's side. Host is left, guest is right for PVP.

Controls

  • Keyboard Controls

      • MTVIE Order inventory (i.e., Tank)
      • H: Extra life 🚁 (N/A in ∞ lives)
      • X Smart missile, Chicken, Banana
      • Space Drop paratroopers
      • Shift Fire gun (or "aimed" missiles)
      • Ctrl or Z Drop bombs
      • ~ or ← Backspace
        Eject pilot + infantry
      • - + Adjust game speed ±5%
      • Esc Show game preferences
      • Pause game
      • Toggle auto-Flip (or double-click)
      • Toggle radar Scaling
      • Enter/return Send a message

  • Mobile Controls

    Controls are always bottom-aligned, and take priority over notifications.

Audio

  • Local ambiance and audible hints - e.g., building convoys, or distant gunfire.

Frame Rate

By default, the game will try to pick the best-performing option.
 Try 30 fps for a more “classic” feel, or if there are performance issues.
The original game ran at perhaps 12-15 fps.

The frame rate setting applies to all players.
⚠️ Network games at 60 fps are experimental.

Game Speed

The game speed setting applies to all players.
⚠️ Network games outside 100% are experimental.

  • Play the game from 10%, to 200% speed. 80% is balanced for touch-based devices. In-game, use - + keys to adjust.
