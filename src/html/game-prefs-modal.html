Frame Rate

By default, the game will try to pick the best-performing option.
 Try 30 fps for a more “classic” feel, or if there are performance issues.
The original game ran at perhaps 12-15 fps.

The frame rate setting applies to all players.
⚠️ Network games at 60 fps are experimental.

Game Speed

The game speed setting applies to all players.
⚠️ Network games outside 100% are experimental.

  • Play the game from 10%, to 200% speed. 80% is balanced for touch-based devices. In-game, use - + keys to adjust.
